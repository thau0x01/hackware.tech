# Hackware — Site Institucional

Site institucional da **Hackware Tecnologia e Inteligência Digital**, construído em HTML, CSS e JavaScript puro para hospedagem via GitHub Pages.

## Estrutura

```
hackware.tech/
├── index.html          # Homepage completa
├── contato.html        # Página de contato com formulário
├── css/
│   └── style.css       # Design system completo + todos os estilos
└── js/
    └── main.js         # Navbar, animações, formulário, contadores
```

## Desenvolvimento local

```bash
# Python 3
python3 -m http.server 8080
# Abra http://localhost:8080
```
